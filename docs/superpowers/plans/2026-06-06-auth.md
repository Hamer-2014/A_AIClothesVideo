# 认证与权限 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 接入 better-auth、Google OAuth、Resend Email OTP/Magic Link，并建立管理员权限 helper。

**Architecture:** better-auth server config 使用 Drizzle adapter；Resend 邮件发送由独立封装 fail-closed；管理员权限由 allowlist + `admin_roles` helper 判断；登录 UI 只暴露 Google、OTP、Magic Link。

**Tech Stack:** better-auth, @better-auth/drizzle-adapter, Resend, Drizzle, Next.js App Router, Vitest.
