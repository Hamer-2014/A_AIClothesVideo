# R2 上传与文件访问 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立私有 R2 上传/下载 signed URL 基础能力，并创建资产记录。

**Architecture:** R2 通过 AWS SDK v3 S3 client 访问；storage key、validation、presign 拆成纯 helper；API route 负责 session 校验和 DB 写入。数据库只保存 R2 key。

**Tech Stack:** Cloudflare R2, AWS SDK v3 S3 client, Drizzle, Next.js App Router, Vitest.
